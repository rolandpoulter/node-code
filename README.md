# Editor

open source, cross platform, runs in a browser, as good as vim/emacs/sublime/cloud9


# Prerequisits

* http://nodejs.org/


# Install

* "npm install gaius -g"
* "gauis run ~/myproject ~/myotherproject"
* open http://localhost:3617
